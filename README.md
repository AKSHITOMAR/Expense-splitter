# 💰 Expense Splitter Web Application

A simple and efficient web-based Expense Splitter built as part of a Hackathon Assessment.

This application allows users to create accounts, add expenses, split them equally among selected participants, and calculate exact settlements (who owes whom).

---

## 🌐 Live Demo

🔗 **Live Application:**  
https://expense-splitter-five-mocha.vercel.app/

---

## 🚀 Features

### 👤 Create Users
- Each user has:
  - User ID (auto-generated)
  - Name

### 💳 Add Expense
Each expense includes:
- Expense ID (auto-generated)
- Description
- Total Amount
- Paid By (one user)
- Participants (selected users)

### ⚖️ Equal Splitting Rule
- The total expense is split equally among selected participants only.

### 📄 View All Expenses
Displays:
- Expense ID
- Description
- Total Amount
- Paid By
- Participants

### 💰 Exact Settlement Calculation
Displays balances in clear format:
- Alice owes Bob ₹250
- Charlie owes Alice ₹100

## ▶️ How to Run the Project

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Start adding users and expenses.

No backend setup required.
