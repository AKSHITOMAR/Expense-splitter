# 💰 Expense Splitter Web Application

A simple and efficient web-based Expense Splitter built for the Hackathon Assessment.

This application allows users to create accounts, add expenses, split them equally among selected participants, and calculate exact settlements (who owes whom).

---

## 🚀 Features

### 👤 Create Users
- Each user has:
  - User ID (auto-generated)
  - Name

### 💳 Add Expense
Each expense includes:
- Expense ID (auto-generated)
- Description
- Total Amount
- Paid By (one user)
- Participants (selected users)

### ⚖️ Equal Splitting Rule
- The total expense is split equally among selected participants only.

### 📄 View All Expenses
Displays:
- Expense ID
- Description
- Total Amount
- Paid By
- Participants

### 💰 Exact Settlement Calculation
Displays balances in clear format:
- Alice owes Bob ₹250
- Charlie owes Alice ₹100
